Electric Thermometer (Arduino Platform)

Minimum system for temperatur detecting. Using Arduino UNO R3 with temperature sensor, 7 segment LED for displaying temperature in 4 scale mode (Celcius, Fahrenheit, Kelvin, Reamur), and LED light for overheat indicator. Collaborate with Mamat Rahmat and Junita Sinambela.

  • Repository: Eletric Thermometer
  • Date: May 2015
  • Service: Platform Development

News Classifier

Web application for classifying news article. Using WEKA 3.6 API for learning and classifying. User input the title and full text of the article which will be classified. The output is the label of the news. User may also give feedback on whether the news correctly classified or not. Collaborate with Luthfi Hamid Masykuri, Annisaur Rosi Lutfiana, Tirta Wening Rachman, and Tony.

  • Repository: News Classifier
  • Date: December 2014
